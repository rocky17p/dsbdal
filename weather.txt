✅ WeatherDataAverage – Full Command Flow
🔹 1. Login
ssh hadoop@10.10.15.238
password: hadoop
🔹 2. Move to working directory
cd $HADOOP_HOME/sbin
mkdir 31451
cd 31451
🔹 3. Create Input File
nano input.txt

👉 Paste your weather data (same as you already used)

cat input.txt
🔹 4. Create Java Program
nano WeatherDataAverage.java

Paste your code.

🔹 5. Compile
javac -classpath $(hadoop classpath) WeatherDataAverage.java
ls
🔹 6. Create JAR
jar cf WeatherDataAverage.jar WeatherDataAverage*.class
ls
🔹 7. Start Hadoop Services (Clean Restart)

⚠️ Always do this cleanly to avoid your earlier errors:

stop-dfs.sh
stop-yarn.sh

start-dfs.sh
start-yarn.sh
🔹 8. Verify Services
jps

👉 Must see:

NameNode
DataNode
ResourceManager
NodeManager
🔹 9. Leave Safe Mode
hdfs dfsadmin -safemode leave
🔹 10. Create HDFS Directories
hdfs mkdir hdfs://localhost:9000/31451
hdfs mkdir hdfs://localhost:9000/31451/input
🔹 11. Upload Input File
hdfs put input.txt hdfs://localhost:9000/31451/input
🔹 12. Remove Old Output (IMPORTANT)
hdfs rm -r hdfs://localhost:9000/31451/output
🔹 13. Run MapReduce Job
hadoop jar WeatherDataAverage.jar WeatherDataAverage hdfs://localhost:9000/31451/input hdfs://localhost:9000/31451/output
🔹 14. Check Output Files
hdfs ls hdfs://localhost:9000/31451/output
🔹 15. View Output
hdfs cat hdfs://localhost:9000/31451/output/part-r-00000
