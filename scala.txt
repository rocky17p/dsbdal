🔹 Factorial (Scala)
Code
def factorial(n: Int): Int = {
  if (n == 0) 1
  else n * factorial(n - 1)
}

val num = 5
println("Factorial = " + factorial(num))
Steps to Run
Open terminal

Create file:

nano factorial.scala
Paste the code → Save (CTRL + O, Enter) → Exit (CTRL + X)

Run using Spark shell:

spark-shell

Inside shell:

:load factorial.scala

Output will be:

Factorial = 120
🔹 Bubble Sort (Scala)
Code
def bubbleSort(arr: Array[Int]): Array[Int] = {
  val n = arr.length

  for (i <- 0 until n) {
    for (j <- 0 until n - i - 1) {
      if (arr(j) > arr(j + 1)) {
        val temp = arr(j)
        arr(j) = arr(j + 1)
        arr(j + 1) = temp
      }
    }
  }

  arr
}

val arr = Array(5, 2, 9, 1, 3)
val sorted = bubbleSort(arr)

println("Sorted Array: " + sorted.mkString(", "))
Steps to Run
Open terminal

Create file:

nano bubblesort.scala
Paste the code → Save → Exit

Start Spark shell:

spark-shell

Load file:

:load bubblesort.scala

Output:

Sorted Array: 1, 2, 3, 5, 9
