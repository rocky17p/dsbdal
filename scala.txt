✅ 1. Word Count using Apache Spark (Scala)
Steps

Create input file

nano input.txt

→ Enter text data and save.

Write Scala program

nano word_count.scala

Add code

val data = sc.textFile("input.txt")

val splitdata = data.flatMap(line => line.split(" "))

val mapdata = splitdata.map(word => (word,1))

val reducedata = mapdata.reduceByKey(_+_)

reducedata.collect

Start Spark Shell

spark-shell

Load the program

:load word_count.scala
Observe output
→ Displays word frequencies as (word, count)
✅ 2. Factorial using Scala
Steps

Create Scala file

nano factorial.scala

Write code

def factorial(n: Int): Int = {
  if (n == 0) 1 else n * factorial(n - 1)
}

val num = 5
println("Factorial = " + factorial(num))

Run using Scala/Spark shell

spark-shell

Load file

:load factorial.scala
Output
→ Displays factorial (e.g., 120)
✅ 3. Bubble Sort using Scala
Steps

Create file

nano bubblesort.scala

Write code

def bubbleSort(arr: Array[Int]): Array[Int] = {
  val n = arr.length

  for (i <- 0 until n) {
    for (j <- 0 until n - i - 1) {
      if (arr(j) > arr(j + 1)) {
        val temp = arr(j)
        arr(j) = arr(j + 1)
        arr(j + 1) = temp
      }
    }
  }
  arr
}

val arr = Array(5, 2, 9, 1, 3)

val sorted = bubbleSort(arr)

println("Sorted Array: " + sorted.mkString(", "))

Run Spark shell

spark-shell

Load file

:load bubblesort.scala
Output
→ Displays sorted array
